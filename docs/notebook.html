Day 1 — 2026-03-10 星期一

实验目的

药用植物活性成分检索与初步筛选

软件/数据库

  • TCMSP数据库（v2.3）
  • PubChem数据库
  • Excel 2021

操作步骤

  1. 登录TCMSP数据库，分别检索 Astragali Radix（黄芪）和 Aconiti Lateralis Radix Praeparata（附子）
  2. 设置筛选阈值：OB ≥ 30%，DL ≥ 0.18
  3. 导出原始成分列表，去重后人工核对

结果记录

Day 2 — 2026-03-11 星期二

实验目的

靶点预测与标准化（基因名转换）

软件/数据库

  • SwissTargetPrediction（Homo sapiens, Probability ≥ 0.7）
  • BATMAN-TCM（Score ≥ 20, adj.P ≤ 0.05）
  • UniProt数据库（基因名校正）
  • R 4.3.2 + org.Hs.eg.db 包

操作步骤

  1. 将32个活性成分导入SwissTargetPrediction批量预测
  2. BATMAN-TCM补充预测低OB但已知药效的成分
  3. UniProt数据库将所有靶点转换为官方Gene Symbol
  4. R语言 clusterProfiler::bitr() 统一基因ID

结果记录

  • SwissTargetPrediction预测靶点：247个（probability ≥ 0.7）
  • BATMAN-TCM补充靶点：89个
  • 合并去重后总靶点：286个
  • ID转换成功率：97.2%（8个靶点无法匹配，手动检索校正）

问题与解决

  • 问题：部分靶点存在多个别名（如AKT1亦称PKB）
  • 解决：以HGNC官方符号为准，建立别名映射表统一

操作步骤

  4. Vina批量对接，exhaustiveness = 32，num_modes = 9

关键相互作用分析

  • Quercetin-AKT1：形成5个氢键（Lys179, Glu198, Asp292, Thr291, Tyr272），π-π堆积（Tyr272）
  • Quercetin-MMP9：4个氢键（Leu187, Ala189, Glu227, Tyr248），螯合催化锌离子
  • Higenamine-TNF：3个氢键（Tyr59, Ser60, Gly121），结合能相对较弱但稳定

结论

所有配体-受体组合结合能均 ≤ -6.5 kcal/mol，其中12组 ≤ -8.0 kcal/mol，验证了网络药理学预测的可靠性。黄酮类成分（槲皮素、异鼠李素）结合能力普遍优于生物碱类（去甲乌药碱）。

实验总结

本研究通过纯计算网络药理学方法，系统解析了芪附汤32个活性成分作用于103个CHF交集靶点的协同机制。核心发现： 1. 核心调控轴：AKT1/IL6/TNF为中心的炎症-凋亡网络 2. 关键通路：HIF-1、PI3K-Akt、TNF三条通路协同性最强 3. 多器官格局：心脏(76.7%) > 肾脏(66.0%) > 肝脏(60.2%) > 血管内皮(55.3%) 4. 药对互补：黄芪偏重于血管生成与代谢调节，附子侧重于心肌收缩与钙信号，二者在炎症与缺氧通路高度协同 5. 对接验证：槲皮素、异鼠李素与MMP9、AKT1、CASP3结合力最强（≤ -9.0 kcal/mol）
