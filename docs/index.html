1 研究概述

1.1 研究背景

慢性心力衰竭（Chronic Heart Failure, CHF）是各种心血管疾病的终末期表现，其病理机制涉及心肌重构、炎症反应、细胞凋亡、血管内皮功能障碍等多维度网络紊乱。传统单一靶点药物难以覆盖CHF的复杂病理网络。

芪附汤（Qifu Decoction, QFD）由黄芪（Astragali Radix）与附子（Aconiti Lateralis Radix Praeparata）组成，为中医”益气温阳”经典药对，临床用于心衰治疗疗效确切。然而其多成分-多靶点-多通路的协同作用机制尚未系统阐明。

1.2 核心科学问题

(1)芪附汤中32种活性成分如何通过多靶点协同作用于CHF病理网络？

(2)药效靶点在心脏、肾脏、肝脏、血管内皮等多器官的分布特征是什么？

(3)黄芪与附子在靶点、通路层面各自的贡献与协同模式如何？

(4)核心活性成分与关键靶点的分子结合模式及亲和力如何？

1.3 研究目标

  • 系统筛选芪附汤活性成分并预测其作用靶点

  • 构建芪附汤干预CHF的”成分-靶点-通路-器官”四维调控网络

  • 定量评估黄芪-附子药对的协同效应

  • 解析核心活性成分与Hub靶点的分子结合机制

  • 为后续实验验证提供计算依据与候选分子

3 详细分析模块

3.1 Module 1: 活性成分筛选与靶点预测

3.1.1 目的

从黄芪与附子中筛选具有良好口服生物利用度和类药性的活性成分，并预测其潜在作用靶点。

3.1.2 数据源与数据库

数据库 版本 用途 官方链接 检索日期
TCMSP v2.3 中药成分检索与ADME筛选 https://old.tcmsp-e.com/ 2026-03-15
SwissTargetPrediction - 靶点预测（概率≥0.7） https://www.swisstargetprediction.ch/ 2026-03-15
BATMAN-TCM - 靶点预测（score≥20） http://bionet.ncpsb.org/batman-tcm/ 2026-03-15
PubChem - 化合物结构与属性验证 https://pubchem.ncbi.nlm.nih.gov/ 2026-03-15
UniProt - 靶点基因名标准化（Homo sapiens） https://www.uniprot.org/ 2026-03-15

3.1.3 筛选标准

  • 口服生物利用度（OB）≥ 30% — TCMSP内置OB值

  • 类药性（DL）≥ 0.18 — 基于Tanimoto系数

  • SwissTargetPrediction概率 ≥ 0.7

  • BATMAN-TCM评分 ≥ 20

  • 物种限定：Homo sapiens

3.1.4 特殊处理

  • 黄芪甲苷（Astragaloside IV）：OB值仅2.22%，但为黄芪标志性成分且药理证据充分，手动补充纳入

  • 靶点基因名统一标准化为HGNC官方符号，去除重复项

3.1.5 预期产出

  • 32种生物活性化合物清单（黄芪13种 + 附子11种 + 共有8种）

  • 286个非冗余靶点（黄芪212个 + 附子188个）

  • 输出文件：Table_S1_32_Bioactive_Compounds.csv

3.2 Module 2: CHF疾病靶点整合与交集识别

3.2.1 目的

整合多源疾病数据库，获取CHF相关致病基因，与药物靶点取交集获得潜在药效靶点。

3.2.2 数据源与数据库

数据库 版本 检索关键词 用途
GeneCards v5.14 “chronic heart failure”, “heart failure” 疾病基因整合（relevance score筛选）
OMIM - “heart failure”, “cardiomyopathy, dilated” 孟德尔遗传疾病基因
DisGeNET v7.0 “chronic heart failure”, CUI: C0018802 疾病-基因关联
TTD - “heart failure”, “cardiac failure” 治疗靶点数据库

3.2.3 分析步骤

  • 分别从4个数据库检索CHF相关基因，合并去重

  • GeneCards按relevance score排序，取前500个

  • 与Module 1的286个药物靶点取交集

  • 韦恩图可视化（VennDiagram R包）

3.2.4 质量控制

  • 基因名统一通过org.Hs.eg.db映射为ENTREZ ID

  • 排除假基因、非编码RNA

  • 交叉验证：至少2个数据库支持的靶点优先

3.2.5 预期产出

  • 103个交集靶标（芪附汤干预CHF的潜在药效靶点）

  • 四数据库韦恩图 + 药对靶点韦恩图

  • 输出文件：Table_S2_103_Intersection_Targets.csv

3.3 Module 3: PPI网络构建与Hub基因筛选

3.3.1 目的

构建蛋白-蛋白相互作用网络，识别网络中的核心节点（Hub基因）。

3.3.2 工具与参数

工具 版本 参数设置
STRING v11.5 organism = Homo sapiens, confidence score ≥ 0.7, 去除孤立节点
Cytoscape 3.9.1 网络可视化与拓扑分析
CytoHubba - 4种算法：Degree, Betweenness, Closeness, ECC

3.3.3 分析步骤

  • 将103个交集靶点导入STRING，获取PPI关系

  • 导出TSV文件，导入Cytoscape

  • 使用CytoHubba插件计算4种拓扑参数

  • 4种算法排名取交集，确定Top 10 Hub基因

  • 按度值大小调整节点大小，构建可视化网络

3.3.4 Hub基因判定标准

  • Degree、Betweenness、Closeness、MCC四种算法排名均进入前20

  • 取并集后按综合排名筛选Top 10

3.3.5 预期产出

  • PPI网络图（节点103个，边数约650条）

  • Top 10 Hub基因：AKT1, IL6, TNF, VEGFA, CASP3, MAPK1, JUN, TP53, MMP9, STAT3

  • Hub基因拓扑参数条形图

  • 输出文件：Table_S2含PPI Degree列

3.4 Module 4: 功能富集分析

3.4.1 目的

系统解析103个交集靶点的生物学功能与信号通路富集特征。

3.4.2 工具与参数

工具 版本 参数
R 4.3.2 分析环境
clusterProfiler 4.8.3 GO/KEGG富集分析
org.Hs.eg.db 3.17.0 人类基因注释库
fgsea 1.26.0 GSEA富集分析
ggplot2 3.4.4 可视化

3.4.3 分析内容

(A) GO富集分析 - Biological Process (BP) — 取Top 50

  • Cellular Component (CC) — 取Top 30

  • Molecular Function (MF) — 取Top 30

  • 显著性阈值：p.adjust < 0.05（BH法校正）

(B) KEGG通路富集分析 - 物种：hsa（Homo sapiens）

  • 取Top 50通路

  • 显著性阈值：p.adjust < 0.05

(C) GSEA基因集富集分析 - 背景基因集：KEGG通路基因集（c2.cp.kegg.v7.5.1.symbols.gmt）

  • 排序方式：按log2FC降序（基于GSE5406心衰vs对照表达谱）

  • 置换次数：1000次

  • 显著性：NES > |1.5| 且 padj < 0.05

3.4.4 3.4.4 可视化方案

  • GO/KEGG气泡图（dotplot）：x=GeneRatio, y=通路名, color=p.adjust, size=GeneCount

  • GSEA富集曲线：核心富集基因标注

  • 富集弦图（可选）：基因-通路关联

3.4.5 预期产出

  • GO-BP 50条、GO-CC 30条、GO-MF 30条、KEGG 50条完整富集结果
  • GSEA显著通路列表（核心富集基因为AKT1, IL6, TNF, VEGFA, CASP3等）
  • 输出文件：Table_S3-S6_GO_KEGG_Enrichment.csv

3.5 Module 5: 多器官定位分析

3.5.1 目的

突破传统单器官（心脏）局限，解析药效靶点在多器官组织的分布特征，揭示心-肾-肝-血管轴协同调控机制。

3.5.2 数据源与工具

数据库/工具 用途
DAVID 基因-组织富集分析
HPA (Human Protein Atlas) 蛋白组织表达验证
BioGPS 基因组织表达谱
Gene ORGANizer 器官水平富集分析
GSE5406 心衰心肌转录组验证（GEO数据集）

3.5.3 分析器官

7个目标器官/组织：

  • 心脏（Heart）— 核心靶器官，含心肌特异性基因验证

  • 肾脏（Kidney）— 心肾轴，肾素-血管紧张素系统

  • 肝脏（Liver）— 药物代谢与炎症因子合成

  • 血管内皮（Vascular Endothelium）— 内皮功能障碍

  • 肺（Lung）— 肺淤血与炎症

  • 脂肪组织（Adipose Tissue）— 脂肪因子与代谢

  • 肾上腺（Adrenal Gland）— 醛固酮与交感神经激活

3.5.4 分析步骤

  • 将103个靶点提交DAVID进行TISSUE富集分析

  • HPA验证核心Hub基因的组织表达特异性

  • Gene ORGANizer进行器官水平富集，计算各器官靶点占比

  • 构建器官-靶点二部矩阵（103基因 × 7器官，二值化）

  • UpSet图可视化器官间靶点共享模式

  • 热图展示Top 30 Hub基因在7器官的分布

3.5.5 预期产出

  • 7器官靶点分布统计（心脏76.7% > 肾脏66.0% > 肝脏60.2% > 血管内皮55.3% …）

  • 多器官热图 + UpSet图

  • 7器官共有核心靶点23个（炎症+凋亡核心节点）

  • 心脏-肾脏特异性共享靶点11个

3.6 Module 6: 四维整合网络与药对协同定量

3.6.1 目的

构建”成分-靶点-通路-器官”四维整合网络，定量评估黄芪与附子的协同效应。

3.6.2 工具

  • Cytoscape 3.9.1 — 二部网络构建与可视化
  • R 4.3.2 — 协同指数计算

3.6.3 网络构建

节点类型（4类）： - 成分节点（32个）：按来源着色（黄芪=蓝，附子=红，共有=紫）

  • 靶点节点（103个）：按度值调整大小

  • 通路节点（Top 20 KEGG）：按富集显著性着色

  • 器官节点（7个）：按靶点数量调整大小

边类型（3类）： - 成分-靶点边（预测关系）

  • 靶点-通路边（富集关系）

  • 靶点-器官边（表达关系）

3.6.4 药对协同定量指标

指标 公式/方法 结果
Jaccard相似系数 共享靶点 / (HQ靶点 ∪ FZ靶点) 0.417
靶点重叠率 共享靶点 / min(HQ, FZ) 0.606
通路协同得分 共享通路基因数 / 通路总基因数 HIF-1: 0.89, PI3K-Akt: 0.85
黄芪优势通路 HQ独有富集通路 VEGF signaling, FoxO signaling
附子优势通路 FZ独有富集通路 Calcium signaling, cGMP-PKG signaling

3.6.5 预期产出

  • 四维整合网络图（分层布局）

  • 药对协同定量分析表

  • 黄芪-附子通路贡献Venn图

3.7.4 受体制备流程

  • 从RCSB PDB下载晶体结构

  • 去除水分子（HOH）

  • 去除原配体/抑制剂

  • 添加极性氢原子

  • 计算Gasteiger电荷

  • 合并非极性氢

  • 保存为.pdbqt格式

3.7.5 3.7.5 配体制备流程

  • 从PubChem下载SDF格式

  • OpenBabel转换为PDB

  • AutoDockTools添加Gasteiger电荷

  • 设置可旋转键（所有非环单键）

  • 保存为.pdbqt格式

4.2 数据源

9 局限性与风险评估

9.1 固有局限性

  • OB/DL筛选偏差：TCMSP的OB模型对皂苷类成分预测偏低，可能遗漏部分活性成分（如黄芪甲苷手动补充）

  • 靶点预测假阳性：计算预测存在假阳性，SwissTargetPrediction概率≥0.7仍有误差

  • 半柔性对接局限：受体刚性假设，未考虑蛋白构象变化；未考虑溶剂效应

  • 组织表达≠功能：多器官定位基于mRNA/蛋白表达，不直接等同于功能活性

  • 缺乏剂量信息：网络药理学为定性/半定量，无法反映成分浓度梯度效应

9.2 潜在风险与应对

风险 影响 应对策略
靶点预测假阳性率高 网络噪声大 多工具交集 + 严格阈值 + PPI过滤
KEGG富集通路过多 结果分散 聚焦Top 20 + GSEA验证 + 器官定位收敛
分子对接结合能偏强 过度乐观解读 结合2D相互作用分析，关注氢键/π堆积质量
数据库更新导致结果差异 可复现性 统一标注访问日期，锁定版本

9.3 论文局限性声明要点

  • 本研究为纯计算预测，所有结论需实验验证
  • 仅纳入口服生物利用度筛选的成分，可能遗漏非口服活性成分
  • 分子对接为静态模拟，不代表真实动态结合过程
  • 未考虑成分间的药代动力学相互作用

10 核心创新点

  • 多器官定位视角 — 突破传统单心脏局限，系统解析心-肾-肝-血管-肺-脂肪-肾上腺7器官靶点分布，揭示心衰多器官受累的调控基础

  • 药对协同定量 — Jaccard指数 + 通路协同得分，定量区分黄芪（益气）与附子（温阳）各自的通路贡献，阐明”相须为用”的分子基础

  • 四维整合网络 — 成分-靶点-通路-器官二部网络，从分子到器官层面全景展示芪附汤调控机制

  • 免疫浸润关联 — CIBERSORTx关联Hub靶点与心肌免疫微环境，补充网络药理学的免疫学维度

  • 多技术交叉验证 — 网络药理学 + 富集分析 + 分子对接三层证据链，提高预测可靠性
